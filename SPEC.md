# 自転車通勤コスト計算ツール【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Cycling Cost Calculator
**バージョン**: 1.1.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/cycling-cost-calculator/

自転車通勤vs電車・車の年間コストを比較計算。健康効果も加味した総合診断。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化
- **入力値保存**: フォーム入力値を自動保存（onchange/oninput）
- **計算履歴**: 最新5件の計算結果を履歴として保存・復元可能

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js 4.4.1（コスト比較の可視化）
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## v1.1.0 新機能（2026-06-10）

### 1. データ保存・履歴機能（M3）
- 入力フォームの値を自動保存（localStorage）
- 計算結果を履歴として保存（最新5件）
- 履歴から前回の計算を復元可能
- 履歴クリアボタンで全削除

### 2. Chart.js による可視化（M4）
- コスト比較（現在の交通手段 vs 自転車 vs 節約額）を棒グラフで表示
- インタラクティブなツールチップ
- レスポンシブ対応（モバイル・PC両対応）

### 3. 印刷・エクスポート機能（M10）
- 📄「印刷」ボタン：window.print() で PDF 化・紙出力
- 📋「コピー」ボタン：結果をテキスト形式でクリップボードにコピー
- 印刷時は広告・フォーム・ナビゲーションを自動非表示

### 4. 目標設定・達成度トラッキング（M6）
- 🎯 目標年間節約額を設定・保存（localStorage）
- 計算結果から達成度% を自動計算
- プログレスバーで視覚的に表示
- 目標未達成時は残額表示、達成時は上回り額を表示
- 設定日時を記録して継続利用を促進

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する
